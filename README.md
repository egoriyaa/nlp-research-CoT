# Nlp-research-CoT

Installation:
```
git clone https://github.com/egoriyaa/nlp-research-CoT.git

cd nlp-research-CoT

pip install -r requirements.txt
```

To generate CoT data run:
```python inference.py```

To calculate metrics for CoT and ensemble CoT run:
```python calculator.py```
